# v1.9.0 릴리스 초안

## Problem

하네스 건강 요약 기능이 여러 PR로 main에 들어갔지만, npm/package/plugin 버전은 아직 v1.8.0에 머물러 있었음.
사용자가 설치나 업데이트로 새 helper, schema, 문서, 리포트 개선을 받으려면 버전 표면과 changelog를 v1.9.0으로 맞춰야 했음.

## Changes

- `package.json`, `package-lock.json`, `.claude-plugin/plugin.json` 버전을 `1.9.0`으로 올림.
- `CHANGELOG.md`의 `[Unreleased]` 하네스 건강 요약 변경사항을 `[1.9.0] - 2026-06-09`로 이동함.
- README/README.ko 상단 최신 패키지 설명과 release badge 링크를 v1.9.0 기준으로 갱신함.
- 하네스 건강 요약, graph overview, run timeline, candidate score, lifecycle state, dormant archive review, 정적 HTML report를 이번 릴리스의 핵심 변경으로 정리함.
